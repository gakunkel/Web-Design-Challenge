Summary: Latitude vs. X

Visualizations

The purpose of this project was to analyze how global weather changes as you approach the equator. To accomplish this, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather in relation to latitude. These aspects included max temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations created as part of the analysis, as well as descriptions of any trends or correlations observed.
